# kDownloader

## How To Use:

1) Download the "kDownloader_V#" batch file.

2) Open the file in a text editor like Notepad, Notepad++, etc.

3) Input your own filepaths for the variables listed at the very top of the file.

4) Save.

5) Double-click to use! :D
